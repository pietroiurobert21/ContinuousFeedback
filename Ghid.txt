Pasi configurare aplicatie:

1. Creati 2 terminale in visual studio code
2. In primul terminal scrieti "cd backend", iar in celalalt "cd frontend"
3. In ambele terminale scrieti "npm install"
4. In folderul "backend", creati un nou fisier ".env"
5. In fisierul ".env" scrieti "ACCESS_TOKEN_SECRET = e3f7"

Rulare:

6. In ambele terminale scrieti npm run dev
7. Serverul de frontend ruleaza pe 	http://localhost:5173/
8. Serverul de backend ruleaza pe 	http://localhost:3000/

Tips:
- dupa ce va logati, daca doriti sa accesati o "activitate" ca si student, accesati calea 	http://localhost:5173/  	in browser
(deoarece nu exista un shortcut/buton catre acest path, odata ce va logati)

-pentru a copia codul activitatii, dati click pe el (se va copia automat in clipboard)
